# YanSimSaveFilemngr
A basic Yandere Simulator Save File manager.

Requirements : .net 4.0

# Features

* infinite money
* set reputation
* set study points
* true pacifist mode - bluescreens the pc when you kill someone
* the ability to export saves to a .reg file
